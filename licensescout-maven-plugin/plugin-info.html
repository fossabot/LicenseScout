Plugin Documentation

Goals available for this plugin:

Goal Description
licensescout:help Display help information on licensescout-maven-plugin.
Call mvn licensescout:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
licensescout:scanJava Scans Java installation directory for licenses.
licensescout:scanNpm Scans NPM package directory for licenses.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.3.0
JDK 11
Memory No minimum requirement.
Disk Space No minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>org.aposin.licensescout</groupId>
          <artifactId>licensescout-maven-plugin</artifactId>
          <version>1.3.1-SNAPSHOT</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>org.aposin.licensescout</groupId>
        <artifactId>licensescout-maven-plugin</artifactId>
        <version>1.3.1-SNAPSHOT</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
